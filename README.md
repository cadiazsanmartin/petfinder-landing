# PetFinder 🐾

Landing page diseñada para promover el rescate y adopción de mascotas.

## Objetivo
Crear una página clara y accesible que motive a las personas a rescatar o adoptar animales, mostrando información y llamados a la acción.

## Diseño UX/UI
El diseño se enfoca en:

- navegación simple
- jerarquía visual clara
- uso de colores cálidos asociados al cuidado animal
- diseño responsive para dispositivos móviles

## Tecnologías utilizadas

- HTML
- CSS
- Bootstrap
